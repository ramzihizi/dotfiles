# Lazysql Cheatsheet

## General

| Key | Action |
|-----|--------|
| `q` | Quit |
| `?` | Show help / keybindings |
| `Backspace` | Return to connection selection |

## Navigation

| Key | Action |
|-----|--------|
| `j` / `k` | Move down / up |
| `g` / `G` | Jump to first / last item |
| `H` | Focus tree / database panel |
| `L` | Focus table display panel |
| `<` / `>` | Previous / next page (table) |

## Tabs

| Key | Action |
|-----|--------|
| `Ctrl+[` | Previous tab |
| `Ctrl+]` | Next tab |
| `X` | Close current tab |

## Query & Editor

| Key | Action |
|-----|--------|
| `Ctrl+E` | Open full-screen SQL editor |
| `Ctrl+R` | Execute query |
| `/` | Focus filter / SQL editor |

## Table Operations

| Key | Action |
|-----|--------|
| `c` | Edit cell |
| `o` | Insert new row |
| `d` | Delete row |
| `R` | Refresh table |
| `K` / `J` | Sort column ascending / descending |
| `Ctrl+S` | Save changes |
