# Lazydocker Cheatsheet

## Global

| Key | Action |
|-----|--------|
| `q` / `Ctrl+C` | Quit |
| `?` / `x` | Open help / options menu |
| `X` | Run custom command |
| `/` | Filter list |
| `Tab` | Cycle between panels |
| `+` / `_` | Next / previous screen mode |

## Panel Selection

| Key | Panel |
|-----|-------|
| `1` | Projects |
| `2` | Services |
| `3` | Containers |
| `4` | Images |
| `5` | Volumes |
| `6` | Networks |

## Navigation

| Key | Action |
|-----|--------|
| `j` / `k` | Move down / up in lists |
| `J` / `K` | Scroll main panel down / up |
| `H` / `L` | Scroll left / right |
| `PgUp` / `Ctrl+U` | Scroll up |
| `PgDn` / `Ctrl+D` | Scroll down |
| `Home` / `End` | Jump to top / bottom |
| `[` / `]` | Previous / next tab |
| `Enter` | Select / focus item |
| `Space` | Toggle selection |

## Containers

| Key | Action |
|-----|--------|
| `r` | Restart |
| `s` | Stop |
| `p` | Pause |
| `d` | Remove |
| `a` | Attach (interactive) |
| `E` | Exec shell into container |
| `m` | View logs |
| `e` | Hide / show stopped containers |
| `c` | Run custom command |
| `b` | Bulk commands |
| `w` | Open in browser |

## Services (Docker Compose)

| Key | Action |
|-----|--------|
| `r` | Restart service |
| `s` | Stop service |
| `S` | Start service |
| `p` | Pause service |
| `u` | Recreate / up service |
| `U` | Up entire project |
| `D` | Down entire project |
| `d` | Remove containers |
| `a` | Attach |
| `E` | Exec shell |
| `m` | View logs |
| `R` | Restart options |
| `w` | Open in browser |

## Images

| Key | Action |
|-----|--------|
| `d` | Remove image |
| `c` | Run custom command |
| `b` | Bulk commands (prune unused) |

## Volumes

| Key | Action |
|-----|--------|
| `d` | Remove volume |
| `c` | Run custom command |
| `b` | Bulk commands (remove unused) |
